# DevOps Git Workflow Demo

This is a simple DevOps project to demonstrate the basic Git and GitHub workflow using Ubuntu terminal.

## 🔧 Steps Covered

1. Configure Git with username and email
2. Clone a GitHub repository using SSH
3. Create a new branch
4. Make changes and stage them
5. Commit changes with a message
6. Push the branch to GitHub
7. Create a Pull Request (PR)
8. Merge changes into `main` branch
