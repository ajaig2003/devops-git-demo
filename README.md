# DevOps Git Workflow Demo

This project demonstrates a basic Git and GitHub workflow using the Ubuntu terminal.

## Steps Covered
1. Set up Git
2. Clone a repository via SSH
3. Create a new branch
4. Make changes
5. Commit and push changes
6. Create and merge a Pull Request
